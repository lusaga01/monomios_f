📖 Introducción al Álgebra

En muchas tareas matemáticas es preciso trabajar con valores de número desconocido o indeterminado. Para ellos, se utilizan letras y se opera con las mismas propiedades que las expresiones numéricas.

Una expresión algebraica es una combinación de letras y números relacionados con las operaciones de suma, resta, multiplicación, división, potenciación y radicación.

🧬 Anatomía del Monomio

El monomio es la expresión algebraica mas simple.

Un monomio es el producto de un número y una o varias letras.

-8xy4
Coeficiente
Parte Literal
  • 📌 Coeficiente: Es el número con su signo. Si no aparece ningún número delante de las letras, el coeficiente es 1. Si sólo aparece el signo menos, el coeficiente es -1.
  • 📌 Parte Literal: Son las letras y sus exponentes.
  • 📌 Grado: Es la suma de los exponentes de sus letras.

    ¿Cómo se calcula? Se suman los exponentes de todas las letras que forman la parte literal. Si una letra no tiene exponente escrito, se entiende que su exponente es 1.

Ejemplos de grado:
• 5x³ → Grado 3
• -2ab² → Grado 3 (porque 1 + 2 = 3)
• 7x → Grado 1

⚠️ Recordatorio: Los exponentes de las letras deben ser números naturales (0, 1, 2...). Si hay una letra en el denominador o dentro de una raíz, no se considera monomio.

🔢 Valor Numérico

Sustituimos las letras por números y resolvemos la operación del monomio. Ejemplo: Para 3x², si x = 2, el valor es 3 · (2)² = 12.

👯 Monomios Semejantes

Tienen la misma parte literal (mismas letras y mismos exponentes).

SÍ 2x² y 5x² son semejantes.
NO 2x² y 2x no son semejantes.

➕ Suma y Resta

Solo se pueden sumar o restar monomios si son semejantes.

Regla fundamental: Para sumar o restar monomios semejantes, se suman o restan los coeficientes y se deja la misma parte literal.

Ejemplo explicado paso a paso:
Operación: 4x² - 2x² = (4 - 2)x² = 2x²

1. Identificar coeficientes: (4) y (-2)
2. Operar números: (4 - 2) = 2
3. Mantener parte literal: x²
Resultado final: 2x²
SÍ 3x + 2x = 5x (sumamos coeficientes).
NO 3x + 2y no se puede sumar (se deja indicado).
